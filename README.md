# KüçükSermayeAvcısı.com - MVP

BIST'te **düşük sermayeli (<500M TL)** ve **düşük fiyatlı (<10 TL)** hisseleri tarar. AI yorumlu screener.

## Özellikler
- Penny stock filtreleme (Fiyat <10 TL)
- Mikro sermaye filtre (Piyasa <500M TL)
- AI yorum (RSI + Hacim bazlı)
- Grafik entegrasyonu (Chart.js)
- PDF rapor (yakında)
- Telegram bot
